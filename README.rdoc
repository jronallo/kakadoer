= kakadoer

Simple wrapper around kakadu_compress to convert TIFs to JP2s. The kakadu_compress
commandline utility must be installed on the system for this to work. See this
{download link}[http://www.kakadusoftware.com/index.php?option=com_content&task=view&id=26&Itemid=22]
for your OS.

== Use

kakado --help
